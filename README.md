# archigraph

Multi-repo code knowledge graphs for AI agents.

`archigraph` indexes a group of related repositories — backend, frontend, mobile, infra — into a unified knowledge graph that AI assistants query through MCP. Cross-repo connections are detected automatically (imports, shared types, HTTP routes, message queue topics, infrastructure resources). Generated documentation augments the graph as queryable narrative bridges.

Built for install-and-forget developer experience: one wizard, watchers self-heal, agents do the work.
